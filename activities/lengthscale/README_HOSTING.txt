SCALE QUEST HK — READY-TO-HOST PACKAGE

This folder is the complete static classroom game. It includes the game,
illustrations, favicon, PDF teacher guide and editable Word teacher guide.
No database or server code is required.

QUICKEST OPTION — NETLIFY
1. Unzip this package.
2. Sign in at https://app.netlify.com/drop
3. Drag the unzipped folder containing index.html into the upload area.

CLOUDFLARE PAGES
1. In Workers & Pages, create a Direct Upload project.
2. Upload this ZIP file directly, or drag the unzipped folder.

GITHUB PAGES
1. Extract these files into the root of a GitHub repository.
2. In Settings > Pages, publish from that repository.
3. Keep index.html and .nojekyll at the repository root.

NOTES
- Student progress is stored only in that browser/device.
- Upload the whole package. The assets and teacher-guide folders are required.
- The game does not collect student names or results.
