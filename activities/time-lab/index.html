Physics Learning Lab • 雙語時間量度活動

Activity 1: Time Scale Match

Focus: Time scales, order of magnitude, scientific notation, and unit conversion.

An interactive matching game designed to be played individually or in pairs. Best used after introducing prefixes and scientific notation.

Activity 2: Pair Investigation

Focus: Human timing variation, repeated measurements, uncertainty, and method design.

A short 10–15 minute Exploratorium-style investigation for student pairs to explore the limits of human reaction time and measurement error.

Technical & Maintenance Information

For website administrators and repository maintainers.

Repository Setup & Installation

Both activities work as standalone HTML files. They do not require a server, build process, external library, or internet connection after download.

  1. In the repository, open or create the folders activities and then time-lab.
  2. Upload all files from this folder without renaming them.
  3. Commit the upload.
  4. On the main homepage, link the Time Lab card to:
<a href="activities/time-lab/index.html">Open Time Lab</a>

On GitHub Pages, the expected address will be:
https://YOUR-USERNAME.github.io/YOUR-REPOSITORY/activities/time-lab/

Maintenance & Licensing Guidelines
  • Keep all links relative so that local use and GitHub Pages both work.
  • When revising an activity, retain the filenames unless the main index is updated at the same time.
  • This folder follows the licence and attribution information of the parent repository; it does not create a separate licence.
