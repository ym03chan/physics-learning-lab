# Time Lab activities This folder contains two self-contained, bilingual time-measurement activities for the **Physics Learning Lab** website. ## Folder structure ```text activities/time-lab/ ├── index.html # Student-facing choice page ├── lab_v1.html # Version 1: Time Scale Match ├── lab_v2.html # Version 2: Time Lab pair investigation ├── teacher-notes.html # Lesson guidance for both versions └── README.md # This file ``` The public paths are therefore: - `activities/time-lab/index.html` - `activities/time-lab/lab_v1.html` - `activities/time-lab/lab_v2.html` - `activities/time-lab/teacher-notes.html` ## Adding the folder to the main repository 1. In the repository, open or create the folders `activities` and then `time-lab`. 2. Upload all five files from this folder without renaming them. 3. Commit the upload. 4. On the main homepage, link the Time Lab card to: ```html Open Time Lab ``` On GitHub Pages, the expected address will be: ```text https://YOUR-USERNAME.github.io/YOUR-REPOSITORY/activities/time-lab/ ``` ## Which version should a teacher choose? | Version | Main learning focus | Suggested use | |---|---|---| | `lab_v1.html` | Time scales, order of magnitude, scientific notation and unit conversion | Individual or pair game after prefixes/scientific notation | | `lab_v2.html` | Human timing variation, repeated measurements, uncertainty and method design | 10–15 minute Exploratorium-style pair investigation | Both activities work as standalone HTML files. They do not require a server, build process, external library or internet connection after download. ## 中文簡介 此資料夾包含兩個雙語時間量度活動。`lab_v1.html` 是時間尺度配對遊戲，著重數量級、科學記數法和單位換算；`lab_v2.html` 是 10–15 分鐘的二人探究，著重人為計時變化、重複量度、不確定度和量度方法設計。 請保留以上檔案名稱和資料夾結構，主頁只需連結至 `activities/time-lab/index.html`。 ## Maintenance - Keep all links relative so that local use and GitHub Pages both work. - When revising an activity, retain the filenames unless the main index is updated at the same time. - This folder follows the licence and attribution information of the parent repository; it does not create a separate licence.
