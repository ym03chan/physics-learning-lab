Design approach 設計理念

Made for real classrooms 為真實課堂而設

Physics before decoration 先重物理，再談裝飾

Visual design supports the model, the measurement, and the student decision. It should not compete with them. 視覺設計應服務於物理模型、測量過程及學生判斷，而非分散注意力。

✓ Clear variables and units 清晰顯示物理量及單位
✓ Immediate, meaningful feedback 即時而有意義的回饋
✓ Pair-work friendly 適合二人合作
✓ Responsive on laptop and iPad 支援手提電腦及 iPad

Bilingual by design 以雙語學習為設計基礎

English and Traditional Chinese are presented as genuine learning modes rather than afterthoughts. The aim is to preserve scientific precision in both languages. 英文及繁體中文並非附加翻譯，而是完整的學習模式；兩種語言均保持科學表述的準確性。

✓ Student-facing instructions 學生版指示
✓ Teacher-ready structure 教師可直接使用
✓ Accessible visual hierarchy 清晰易讀的視覺層次
✓ No installation required 毋須安裝
